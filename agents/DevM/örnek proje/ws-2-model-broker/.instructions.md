# DevM Workspace - VS Code Copilot Talimat Sistemi

Bu workspace DevM platformunun bir parçasıdır.
AI motoru: **VS Code Copilot Agent**.

## TALIMATLAR.md Kuralları

- `TALIMATLAR.md` dosyasındaki `- [ ]` ile başlayan satırlar uygulanmayı bekler.
- `- [x]` olanlar tamamlanmış, atlınır.
- Bir talimatı uyguladıktan sonra `- [ ]` → `- [x]` olarak güncelle.

## Görev Formatı

```
- [ ] Kısa açıklama
  - Detay veya bash bloğu altında olabilir
```

## Dosya Sahipliği

Her workspace sadece kendi `proje/` klasörüne yazar. Diğer workspace klasörlerine dokunma.

## Commit Kuralı

- Bir talimat grubunu uyguladıktan sonra değişiklikleri git commit et.
- Commit mesajı: `feat: TALIMATLAR - <özet>`
